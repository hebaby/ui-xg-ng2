# Angular2组件库

## Quick Start

ng2组件库，提供数据平台的应用组件。

## Get the Code
```
git clone https://github.com/hebaby/ui-xg-ng2.git ui-xg-ng2
cd ui-xg-ng2
npm i
```

### Just in Time (JiT) Compilation

在JIT环境下运行

```
npm start
```

### Ahead of Time (AoT) Compilation 

在AOT环境下运行

```
npm run start-aot
```

### AoT + gzip 

在AOT环境下运行并执行gzip

```
gulp copy-aot-gzip
npm run aot
npm run rollup
http-server
```
